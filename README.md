## <アプリ名>
  Medi Navi(メディナビ)
  
  URL https://medi-navi.site/
### サイト概要
  病院・クリニックの検索、口コミ投稿、スケジュール管理、会員同士のコミュニケーションができるサイト。

### サイトテーマ
  口コミと評価で病院・クリニックを検索、スケジュール管理できるサイトを作成。

### テーマを選んだ理由
  既存のサイトではスケジュール管理ができる機能があまり見受けられない。
