# <アプリ名>
  Medi Navi(メディナビ)
## サイト概要
  病院・クリニックの検索、口コミ投稿、スケジュール管理ができるサイト。

### サイトテーマ
  信頼できる口コミと評価で病院・クリニックを検索、スケジュール管理できるサイトを作成する。

### テーマを選んだ理由
  既存のサイトではスケジュール管理ができる機能があまり見受けられない

### ターゲットユーザ
　病院・クリニックを利用したいユーザー

### 主な利用シーン
　通院したいがどこの病院が良いのか、病院の場所等の情報が分からない時
